# RMS Web Front End

브릿지 RMS(Robot Management System) 웹 프론트 엔드 프로젝트

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
